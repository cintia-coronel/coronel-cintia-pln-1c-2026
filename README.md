Este repositorio reúne los trabajos prácticos, ejercicios de clase y proyectos integradores desarrollados a lo largo de la materia.

## 🛠 Habilidades Desarrolladas
Durante el desarrollo de los proyectos, trabajamos las siguientes competencias clave:

- Manipulación de Datos: Extracción, limpieza y normalización de datasets complejos utilizando Python y librerías especializadas. 🐍

- Análisis de Lenguaje: Aplicación de técnicas para explorar, interpretar y categorizar el discurso humano a través de modelos computacionales. 🔍

- Arquitecturas RAG: Implementación de sistemas de recuperación de información aumentada por generación (Retrieval-Augmented Generation) para conectar datos locales con modelos de lenguaje.

- Visualización: Representación gráfica de hallazgos para facilitar la toma de decisiones basada en datos.
